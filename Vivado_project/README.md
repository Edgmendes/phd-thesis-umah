# phd-thesis-umah
Suplementary Material Eduardo Mendes PhD Thesis


Version of Vivado v2018.2.
The board for which the project was done TE0745. Board files should be added to Vivado installation folder.  E.g., C:\Xilinx\Vivado\2018.2\data\boards\board_parts.
Vivado Licenses can be found on Lincense/ and should be added.
Constraint files are used for I/O pin mapping and are specific to our Ultrasonic Mid-Air prototype.

The Vivado project is in Project/. The IPs of the project can be found on ip_repo. And the project is TE0745v18-Haptic/Haptic_prj_18bits
