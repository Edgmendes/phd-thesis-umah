# phd-thesis-umah
Suplementary Material Eduardo Mendes PhD Thesis

This is directory contains the code developed for processing the results of the user studies realized in the thesis.
The first user study dealt with the specification of the haptic sensations. And the second user study regarded
the haptic sensations (including combined haptic sensations) selective identification. 
The results of the Specification study were coded in Matlab. And the results of the Selective Identification were coded in
Python.
