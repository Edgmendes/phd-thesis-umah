# phd-thesis-umah
Suplementary Material Eduardo Mendes PhD Thesis

	- Haptic Sensation Tool;
	- Code developed for the acoustic measurements;
	- Documentation and Results of the Perceptual User Studies;
	- Vivado project developed for the Ultrasonic Mid-Air Haptic Prototype.
